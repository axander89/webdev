HTML Basic Structure

Every HTML file is made up building blocks called tags, its corresponding attributes, and the enclosed data. It is important to know that some tags are block-level elements. This means that they have their own 'box' or 'line'. In contrast, in-line elements don't have their own 'box' or 'line', but they share the line among other in-line elements.

These building blocks are orginized in the standard format as follows:

<!DOCTYPE html>
<html>

    <head>
      <title> </title>
    </head>

    <body>
    </body>

    <footer>
    </footer>

</html>
