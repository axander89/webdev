CSS Basics Structure

Cascading Style Sheets (CSS) is in escense what gives style to html files. CSS can thought of as the "adjective" of html.

CSS is structured in the standard format as follows:

* {
    all_property_1: value;
    all_property_2: value;
}
element_selector {
    element_property_1: value;
    element_property_2: value;
}
.class_selector {
    class_property_1: value;
    class_property_2: value;
}
#id_selector {
    id_property_1: value;
    id_property_2: value;
}
ancestor_element decendant_element {
    decendant_property_1: value;
    decendant_property_2: value;
}
element adjacents {
    adjacents_property_1: value;
    adjacents_property_2: value;
}
element[attribute='xyz_attribute'] {
    attribute_property_1: value;
    attribute_property_2: value;
}
element:nth-of-type(n) {
    nth-of-type(n)_property_1: value;
    nth-of-type(n)_property_2: value;
}


Note! You should be aware of Specificity. Specificity refers to the priority a CSS selector is given.

color property

  • named-color values

    • color: tan;
    • color: rebeccapurple;
  • hex-color values
    • color: #090;
    • color: #009900;

    • color: #009900aa;
  • rgb() values
    • color: rgb(34, 12, 64, 0.6);
    • color: rgba(34, 12, 64, 0.6);
    • color: rgb(34 12 64 / 0.6);
    • color: rgba(34 12 64 / 0.6);
    • color: rgb(34 12 64 / 60%);
    • color: rgba(34 12 64 / 60%);
  • hsl() values
    • color: hsl(30, 100%, 50%, 0.6);
    • color: hsla(30, 100%, 50%, 0.6);
    • color: hsl(30 100% 50% / 0.6);
    • color: hsla(30 100% 50% / 0.6);
    • color: hsl(30.0 100% 50% / 60%);
    • color: hsla(30.0 100% 50% / 60%);

The Box Model

This is a model in which every element in HTML is a box with four main elements:

  • Content
  • Padding
  • Border
  • Margin
